O Windows

Windows je operační systém, který změnil způsob, jakým lidé komunikují s technologiemi. Od svého uvedení v roce 1985 Windows neustále inovuje, aby přinesl moderní řešení pro práci, zábavu i každodenní život.

🌟 Windows Aero (2006)

Představené průhledné, skleněné okna pro revoluční uživatelské rozhraní. Více na Windows Vista.

📶 Virtuální plochy (2015)

Podpora produktivity umožněním více desktopových nastavení bez námahy. Přečtěte si více na Virtual Desktop.

🤖 Cortana AI asistentka (2014)

Váš osobní digitální asistent, který se učí a přizpůsobuje vám. Další informace na Cortana.

🔒 Windows Hello (2015)

Revoluční přihlášení pomocí biometrie – bezpečnost a pohodlí v jednom. Čtěte více na Windows Hello.

Časová osa inovací

  • 1985: Uvedení Windows 1.0.
  • 1995: Windows 95 přináší nabídku Start.
  • 2001: Windows XP – revoluce v uživatelském zážitku.
  • 2012: Windows 8 – první dotykově orientovaný systém.
  • 2015: Windows 10 – sjednocení platformy.
